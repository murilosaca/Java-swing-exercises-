# Java-swing-exercises-
A collection of Java exercises built with Swing (GUI) and developed in NetBeans, using Maven for project management.

Projects


T08EX01 – Simple name registration form
T08EX02 – Liters converter
T08EX03 – Academic system (students, teachers, subjects) with classes Aluno, Professor, Pessoa, and Disciplina


Tech Stack


Java 8
Swing (GUI)
Maven


How to Run

Each exercise is a standalone Maven project. To build and run one:

bashcd T08EX01
mvn clean compile
mvn exec:java -Dexec.mainClass="t08e01.Nome"

(Adjust the folder and main class name for each exercise.)

Notes

These were built as practice exercises for an Object-Oriented Programming course.
